## About Qawama

Qawama (قواما) is a personal finance app that automatically tracks, categorizes, and analyzes your transactions. The name derives from Surah Al-Furqan 25:67, meaning balanced and moderate spending.

**Key features:**
- Automatic SMS reading from Omani banks (NBO, Bank Nizwa, Bank Muscat, and more)
- Smart categorization with a learning engine
- Budget tracking with alerts
- Financial insights and spending trends
- Full Arabic language support with RTL layout
- Privacy-first — all data stays on your device
- Optional encrypted cloud backup
